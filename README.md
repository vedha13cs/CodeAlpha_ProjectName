# 🚀 Python Mini Projects Collection

A collection of beginner-to-intermediate Python projects designed to demonstrate problem-solving, automation, and programming fundamentals. This repository showcases practical applications of Python using core concepts such as loops, dictionaries, file handling, regular expressions, and object-oriented thinking.

## 📋 Projects Included

### 🎯 1. Developer Hangman Challenge

A console-based word guessing game inspired by the classic Hangman game.

#### Features

* Programming-themed words
* Hint system
* Score tracking
* Input validation
* Replay functionality
* Clean console interface

#### Concepts Used

* Lists
* Strings
* Loops
* Conditional Statements
* Random Module
* User Input Handling

---

### 📈 2. Smart Portfolio Tracker

A simple stock portfolio management application that calculates investment value using predefined stock prices.

#### Features

* Track multiple stocks
* Calculate total investment value
* Portfolio summary report
* CSV report generation
* Error handling for invalid stock symbols

#### Concepts Used

* Dictionaries
* Loops
* Arithmetic Operations
* File Handling
* CSV Module

---

### 📧 3. Email Extractor & Report Generator

A task automation tool that extracts email addresses from a text file and saves them into a separate report.

#### Features

* Email extraction using Regular Expressions
* Duplicate email removal
* Report generation
* File reading and writing
* Exception handling

#### Concepts Used

* Regular Expressions (re)
* File Handling
* Sets
* String Processing
* Exception Handling
