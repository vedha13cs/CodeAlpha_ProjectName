# 📧 Email Extractor & Report Generator

A Python automation tool that extracts all email addresses from a text file and saves them into a separate file.

## Features

- Extracts emails using Regular Expressions
- Removes duplicate emails
- Generates extraction report
- Saves results to a text file
- Handles missing file errors

## Technologies Used

- Python
- Regex (re)
- File Handling

## Run

```bash
python extractor.py
```

## Input

A text file containing any text and email addresses.

## Output

- Console Summary
- extracted_emails.txt

## Learning Concepts

- Regular Expressions
- String Processing
- Exception Handling
- File Operations
