# 📈 Smart Portfolio Tracker

A Python-based stock portfolio management system that calculates total investment value using predefined stock prices.

## Features

- Add multiple stocks
- Quantity tracking
- Investment calculation
- Portfolio summary
- CSV report generation
- Input validation

## Technologies Used

- Python
- Dictionary
- Loops
- File Handling
- CSV Module

## Available Stocks

| Stock | Price |
|---------|---------|
| AAPL | $180 |
| TSLA | $250 |
| GOOGL | $140 |
| MSFT | $420 |
| AMZN | $190 |

## Run

```bash
python portfolio_tracker.py
```

## Output

- Console Portfolio Summary
- CSV Report Generation
